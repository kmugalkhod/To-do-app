# Assets

Replace these placeholder files with your actual app icons:

## Required Assets

1. **icon.png** (1024x1024) - App icon for iOS and Android
2. **splash.png** (1284x2778) - Splash screen image
3. **adaptive-icon.png** (1024x1024) - Android adaptive icon foreground
4. **favicon.png** (48x48) - Web favicon

## Quick Start

You can generate these using online tools like:
- https://easyappicon.com
- https://appicon.co
- https://makeappicon.com

Or use Expo's icon generator:
```bash
npx expo-optimize
```

## Temporary Solution

Until you add custom icons, Expo will use default placeholders.
The app will still work fine for testing.
